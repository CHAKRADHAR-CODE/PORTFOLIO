# Personal Portfolio Project

## Project Overview

This is a modern, high-performance **personal portfolio website** built to showcase my skills, projects, and professional profile in a clean and impactful way.  
The goal of this project is to create a strong first impression for recruiters and interviewers through design, animations, and structured content.

## Features

- Smooth and professional UI animations
- Fully responsive design (desktop, tablet, mobile)
- Clean and scalable code structure
- Fast loading and optimized performance
- Modern UI components and layouts
- Recruiter-focused content presentation

## Tech Stack Used

- **Vite** – Fast development build tool
- **TypeScript** – Type-safe, scalable JavaScript
- **React** – Component-based UI architecture
- **Tailwind CSS** – Utility-first styling
- **shadcn/ui** – Clean and accessible UI components

## How to Run This Project Locally

### Prerequisites
Make sure you have the following installed:
- Node.js
- npm

### Steps to Run

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project folder
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
